# AWS Internal Case Management System

An internal case reporting and processing platform built with Flask, MySQL, and AWS.

## Project Overview
This project is designed as an internal case management system for operational teams.
It supports role-based workflows for submitters, agents, and administrators.

## Core Features
- Case creation and tracking
- Role-based access control
- Search and status filtering
- RESTful API
- AWS EC2 deployment
- MySQL database
- Planned S3 attachment storage and basic monitoring

## Tech Stack
- Python
- Flask
- MySQL
- HTML / CSS
- AWS EC2
- Amazon S3
